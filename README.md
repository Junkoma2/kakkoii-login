# kakkoii-login

ログイン演出のデモアプリです。実際の認証は行いません。

## 概要

メールアドレスとパスワードを入力してログインボタンを押すと、ハッカー風のターミナル演出が流れ、成功画面に切り替わります。

## 注意

このアプリはUI演出のデモです。入力した情報は外部に送信されず、認証機能もありません。

## 操作方法

1. メールアドレスとパスワードを入力（形式チェックのみ）
2. 「LOGIN」ボタンを押すとローディング演出が始まる
3. 成功画面で「LOGOUT」を押すと最初の画面に戻る
